## 功能
- [x] 保存你的进度（在你的本地设备上），即使在更新/补丁后也能保留它
- [x] 根据标题/描述（例如 `"跟踪"`）和版本（`"1"` 或 `"1.5"`）进行搜索
- [x] 自动过滤存在冲突的成就
- [x] 零数据收集，没有跟踪器和广告。
